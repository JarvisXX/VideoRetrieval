Train路径下为基于轨迹的异常行为检测的训练部分，训练视频在Video/Train里。
训练完成后在Video路径下生成四个文件储存聚类信息。
训练代码中有两处system("pause");如有不便可以删掉。

VideoRetrieval路径下为级联的测试部分，完成后在Video路径下输出item_data.txt文件，即有关目标的数据。

目前能从视频中提取出的数据有：
1、[ID]目标的ID。
2、[V ID]目标所在的视频ID(出自哪个视频)。
3、[ID inV]目标在视频中的ID(在视频中第几个出现)。
4、[Cate]目标的类型(0-车辆，1-行人)。
5、[Color1,2]目标衣着的颜色(0-NULL，1-橙色，2-黄色，3-绿色，4-紫色，5-红色，6-白色，7-灰色，8-黑色，9-蓝色)。
6、[SFrame]目标轨迹的起始帧。
7、[EFrame]目标轨迹的结束帧。
8、[Ab V]目标是否有速度异常(1-异常，0-无异常)。
9、[Ab Traj]目标是否有轨迹异常(1-异常，0-无异常)。
